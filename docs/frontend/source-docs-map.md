# 프론트에게 필요한 기존 SA/백엔드 문서 매핑

## 반드시 봐야 하는 문서
| 기존 문서 | 프론트 활용도 |
|---|---|
| `01_프로젝트_개요서.md` | 서비스 목표, 사용자 가치, 플랫폼 방향 이해 |
| `02_사용자_시나리오.md` | 사용자/상인/관리자 페르소나와 핵심 흐름 이해 |
| `03_유스케이스_명세서.md` | 화면별 기능 범위, MVP 여부, 액터별 흐름 확인 |
| `04_기능_명세서.md` | 기능별 입력값, 검증, 비즈니스 규칙 확인 |
| `06_API_명세서.md` | API endpoint, 요청/응답 확인 |
| `12_공통_에러코드_설계서.md` | 에러 응답 포맷, 사용자 메시지 처리 기준 |
| `local-development.md` | 로컬 API 서버, CORS, 쿠키 인증 설정 |
